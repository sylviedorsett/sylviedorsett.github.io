Rain or Shine USA

Site Plan

Domain Name

The site name will be called rainorshineusa.com. It will cost $12.95 per year.

Site Purpose

The main purpose for rainorshineusa.com is to inform users about the current weather for three towns: Preston, Soda Springs and Fish Haven. The content will also include images and climate information for all three towns. The images will reflect the current weather conditions. There will be a Storm Center page which will track and update users on severe weather.
